#### 1. 下载Python

```

```

#### 2. vscode 基本配置

配置解释器**cmd+shift+p**，如下图
![](img/1.jpg)

选择相关解释器后，会生成.vscode文件settings.json**
![](img/2.jpg)
```
// settings.json文件内容
{
    "python.pythonPath": "/usr/local/bin/python3"
}
```

> 但是不如直接在vscode 用户配置目录统一配置
